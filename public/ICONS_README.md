# PWA Icons

The following icons need to be created and placed in the `/public` directory:

## Required Icons

1. **icon-192.png** (192x192 pixels)
   - Used for Android home screen
   - Should be the app logo with sufficient padding

2. **icon-512.png** (512x512 pixels)
   - Used for splash screen
   - Should be the app logo with sufficient padding

## Existing Icons (already in place)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png

## Optional Screenshots (for better install prompt)
- screenshot-map.png (1280x720) - Map page with buses
- screenshot-station.png (750x1334) - Station departure times

## Temporary Solution
Until proper icons are created, you can:
1. Use existing favicons scaled up
2. Use a simple colored square with a bus emoji
3. Generate icons using https://realfavicongenerator.net/

## Icon Design Guidelines
- Use the app's primary color (#2563eb - blue)
- Include a bus icon or Porto landmark
- Ensure good contrast
- Test on both light and dark backgrounds
- Add safe zone padding (10% minimum)
